在美国加州，从旧金山沿着著名的一号公路（Highway One）往南至洛杉矶，或者从洛杉矶往北至旧金山，一路蜿蜒，数百公里的海岸公路，举目望去都是如史诗般令人窒息的壮美。从一号公路的蒙特雷（Monterey）途径大瑟尔（Big Sur）一直到圣路易斯奥比斯波(San Luis Obispo)，全长129公里，被美国交通部“国家风景道路项目”和加利福尼亚州分别誉为“泛美风景大道”（All-American Road）和“景观公路”（State Scenic Highway）。宛若天堂般的海岸风光成为自驾游游客心目中首屈一指的黄金线路。
如果从中央山谷（Central Valley）出发，取道156号公路（Highway 156）往西就能欣赏到一号公路最美的一段海滨风景线。在一号公路的沿线分布着许多观景点，那里是停车游玩最完美的地方。不再迷失于城市的人行道和黄色斑马线之间，让我们从一号公路出发，往南一路驰骋，去追寻蒙特雷最美丽的风光，如果从一号公路往北，则行程恰好相反。
一号高速公路最北端的起点位于旧金山的金门大桥（Golden Gate Bridge），依次穿越旧金山、军营国家公园（ark Presidio）、金门大桥公园（Golden Gate Park）直至第19大道（19th Avenue）。汇入洲际280号线（I-280）后途经数公里便可抵达戴利城（Daly City），随后的一号公路与洲际公路分离，一路往西，绵延成一道令人凝神屏息的旖旎风光。清晨，第一缕晨曦划破黎明，正是出发的最好时光，不仅仅是为了避免交通堵塞，更重要的是沿着海岸线一路驱车去找寻更多的风景。
跨越亚利桑那州的圣克鲁斯县边境便进入风光秀美的海滨城市蒙特雷郡，临近太平洋的一号公路蜿蜒曲折，一路向西。可以在莫斯兰丁州立海滩（Moss Landing State Beach）稍作停留，自由呼吸海风吹来的潮润空气，观赏四周野生动物悠闲的生态和可爱的模样。前行至滩海路（Jetty Road）右转，沿着弯曲的公路在埃尔克霍恩（Elkhorn Slough）河口和沙丘之间来来回回，上上下下，起伏跌宕，宛如梦游仙境。
在路的尽头，车道的左手边有一个停车场供游人休憩。从停车场向一号公路望去，可以看到公共码头，停车场的正前方，靠近码头的地方许多憨态可掬的小海狮一动不动地晒着太阳。而在左边的沙滩上，海豹一扭一扭地来回走着；栖息在前方靠近右边的岩石上的是著名的鹈鹕，而活泼好动的水獭则在海水中嬉戏欢腾。如果幸运地话，或许还能看到苍鹰、白鹭、鸬鹚以及其他水鸟。自带午餐的朋友可以坐在附近的野餐桌上一边欣赏野生动物最原生态的表演，一边在轻轻吹拂的海风中享用快乐的午餐。
途径洋蓟地之后，进入一片如山般的沙丘区，码头州立海滩（Marina State Beach）近在眼前。抬头仰望，天空低垂，和茫茫沙丘连成一片。微风拂面而来，头顶上几架滑翔机划过天空，留下一道道美丽的弧线。想坐下来，静静欣赏宁静的美景，就在“救赎之路”（Reservation Road）下车，一路步行至海滩。诚然，滑翔机并不是一号公路上最后的飞行者，再往南一点，还能看到天空中飘荡着许多拱形、色彩缤纷的降落伞。滑翔伞运动员们喜欢在距离海滩悬挂式滑翔机飞翔以南数英里的地方进行滑翔运动，追逐心中的自由之梦。在不远的天空中，还可以依稀看到中国龙和鲨鱼造型的各式风筝，码头州立海滩是蒙特雷郡最适合放风筝的地方。
跨越边境进入蒙特雷市（City of Monterey）后，在太平洋丛林市/德尔蒙特大道（Pacific Grove/Del Monte Avenue）出口驶离一号公路。沿着德尔蒙特大道行驶约一英里多点路程，透过车窗，就能看到浩瀚无边的大海，不一会儿宁静迷人的老渔人码头出现在眼前。半个多世纪以来，老渔人码头一直是蒙特雷顶级旅游目的地之一。沿着海岸线走向码头的尽头，一排排露天小吃店逐一排开，可以免费品尝到新鲜美味的蛤蜊浓汤。看海水与陆地相接的地方，慵懒的水獭、海狮和海豹躺在茫茫的沙滩上尽情享受加州最灿烂的阳光，任凭波涛拍岸而来。
远处，帆船点点，不时一只贪玩的小海豚横空而出，一个漂亮的腾空翻引来游人驻足观望。在离开之前，别忘了去老渔人码头附近的海关大楼广场（Custom House Plaza）、蒙特雷州立历史公园（Monterey State Historic Park）以及四周的历史建筑物周围漫步，亲自体验蒙特雷的早期历史。海关大楼（Custom House）是加利福尼亚州最早的地标性建筑，也是加州第一座砖房，象征着加州的正式诞生。
著名的卡梅尔艺术小镇上有三家不同风格的酒店，带来不一样的体验。第一家位于左手边的“卡梅尔米迅旅馆”（ Carmel Mission Inn）是一家时尚潮流的精品酒店。喜欢文艺范一点的朋友，右手边的“卡梅尔河旅馆”（ Carmel River Inn），气氛更加悠闲，这是一座位于风景如画的卡梅尔河流旁的农舍酒店，四周树木繁茂，郁郁葱葱。喜欢豪华风格的，可以沿着一号公路前行大约一英里左右就能看到“卡梅尔高原凯悦酒店”（Hyatt Carmel Highlands），这也是《康德纳斯特旅游者》（Condé Nast Traveler）杂志黄金榜和《世界旅游+休闲》（Travel + Leisure's World）杂志最佳排行榜反复提到的一家豪华型酒店。
罗伯斯角州立保护区，通常又被称作“州立公园系统的皇冠之珠”。在这里购买门票后，接下来的一整天可以免费去任何一家州立公园，这可算得上是自驾游穿越大瑟尔最值得投资的好项目！将车停在海狮观景点（Sea Lion Point）停车区，沿着柏树林（Cypress Grove）步道往下走，经过短暂的徒步旅行，就能领略到罗伯斯角全部的自然美景。
从罗伯斯角的南边驱车数英里，抵达加州市场（California Market）后，停下来吃早餐，一边欣赏撼人心扉的风景。坐在露天阳台上，可以将不远处的海岸风光一览无余，外面海风不时吹过，稍带寒意，走进室内，坐在邻近窗户的地方，风景同样让人心驰神往。
继续从岩石角（Rocky Point）驱车往南，接下来将穿越大瑟尔最美丽的秘境之一-- 格拉帕塔州立公园（Garrapata State Park）。公园入口处只有一个非常小的棕色标志，然而从一号公路边缘驶过的一辆接着一辆的小汽车足以让你能够找到每一条公园道路的入口。虽然看起来和一般的高速公路有所不同，其实也只需要走几步就能看到位于公路西边附近的一条公园小径。在那里，可以尽情欣赏海边悬崖峭壁和潮涨潮落壮丽的景色。
接着朝南前行，一号公路沿线有两处停车观景点，是人们拍照留念最多的风景胜地。第一个是大名鼎鼎的比克斯比大桥（Bixby Bridge）。雄伟壮观的比克斯比大桥，是美国西海岸最上镜的大桥之一，经常出现在歌曲、照片和众多汽车商业广告之中，已经成为加州的显著地标。第二个游人常常驻足的地方便是著名的瑟尔角（Point Sur），在离海岸不远的地方一块巨大的火山岩石，顶端110米处矗立着一座历史悠久的灯塔。灯塔每周六、日和星期三对外开放，供游人参观。
再往下，一号公路的下一个公园同样是以绝世风景闻名于世的安德鲁莫乐拉州立公园（ Andrew Molera State Park）。将车停在停车场，然后沿着海滩步道（Beach Trail）前往纯净无瑕的美丽海滩后再重新折回停车场，来回也只有两英里，整段路程轻松而又惬意。
从这里开始，一号公路又开始向内陆延伸数英里，道路两旁成排的红杉树突然映入眼帘。不时还可以看到一些商店、艺术家画廊和餐馆。与大瑟尔河旅馆（Big Sur River Inn）相邻的“一号工作室”（Studio One ）是当地最好的艺术画廊之一。这里还有许多适合停下来享受美味午餐的好地方，想体验正宗的大瑟尔当地美食，可以前往“ 奈裴斯”（Nepenthe）餐厅用餐。这里所有的座位都面朝太平洋海岸，食客坐在长长的餐桌旁，与美食美景同醉。
距离“奈裴斯”餐厅以南不远处，在公路的东面有一家名叫“亨利米勒图书馆”（Henry Miller Library）的书店，这是大瑟尔著名的画廊和社区活动中心。整栋建筑坐落在红杉树树林之中，在这里拾一本书架上的书籍阅读，喝一杯咖啡，享受难得的午后时光，再接着上路。此行大瑟尔最后一处必去的拍摄点是位于茱莉亚费佛州立公园（Julia Pfeiffer Burns State Park）的麦克维瀑布(McWay Falls) 。沿着停车场的瀑布步道（Waterfall Trail ）往西走一小段路程就能看见绿水环绕的麦克维瀑布，这里是大瑟尔上镜率最高的景点之一，留给人们难以忘怀的美丽倩影。
想在大瑟尔度过一个真正独特的夜晚，可以在茱莉亚费佛州立公园以南25英里处的“三骨度假村”（Treebones Resort）预订一座圆顶帐篷。这座生态友好型度假村利用可再生资源为游泳池、浴缸中的水加热，为所有的建筑供暖。融四周美景于一体，和谐相生，为游客带来了原生态的舒适感。而度假村室内餐厅采用的蔬菜全部自产自足。
